Generator函数

yield 和 return

  • yield 返回的done状态为false
  • return 返回的done状态为true,并不执行return后面的语句

for…of 循环

  • 一旦next方法的返回对象的done属性为true，for…of循环就会中止，且不包含该返回对象，所以上面代码的return语句返回的6，不包括在for…of循环之中。
  • 扩展运算符（…）、解构赋值和Array.from方法内部调用的，都是遍历器接口。
