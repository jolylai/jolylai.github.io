    • Linux操作命令

    [root@localhost ~]

    • root 当前登录用户
    • localhost 主机名
    • ~ 当前所在目录(家目录)

    • 超级用户提示符

    • $ 普通用户提示名

    命令格式

    • 命令 [选项] [参数]

    • ls [选项] [文件或目录]

    • 选项
      -a 显示所有文件,包括隐藏文件
      -l 显示详细信息
      -d 查看目录属性
      -h 人性化显示文件大小
      -i 显示inode

    -rw-r–r-## 1

      • 文件类型
    • rw- u所有者
    • r– g所属组

    • r– o其他人

    • r 读权限

    • w 写权限
    • x 执行

    文件处理命令

    • 递归创建
      mkdir:make directories

    切换目录

    • cd [目录]
    • change directory
    • 简化操作

    删除文件或目录

    • rm -rf[文件或目录]
      rm:remove directory
      -r 删除目录
      -f 强制

    • rmdir:删除空文件
      rmdir japan

    rm:remove directory
    rm -rf [文件]
    rm -rf japan/

    复制

    • cp [选项] [原文件或目录] [目标目录]
      选项
      -r 复制目录
      -p 连带文件属性复制
      -d 若原文件是连接文件，则复制连接属性
      -a 相当于 -pdr
      cp -r japan/ /temp/

    剪切或改名

    • mv [原文件或目录] [目标目录]
